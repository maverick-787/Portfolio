# About
A personal portfolio website built using HTML5 and CSS3.

## File Structure
- index.html — main webpage (structure and content)
- style.css — all styling, animations and responsive design

## Sections
- Home — introduction and hero section
- About — background, skills and tools
- Projects — showcase of previous work
- Contact — email and social media links

## Features
- Responsive design (works on mobile and desktop)
- Smooth scroll navigation
- Hover animations on cards and buttons
- Fade-up animation on page load
